This library provides the ability to include visually distinct and appealing descriptive blocks
with information in your `typst` files.

The syntax embeds seemlessly in your markup.

For further information, please have a look at the [example.pdf](https://gitlab.kit.edu/marc.thieme/theorems/-/jobs/artifacts/main/file/examples/example.pdf?job=compile_example)
and its [source code](https://gitlab.kit.edu/marc.thieme/theorems/-/blob/main/examples/example.typ?ref_type=heads).
